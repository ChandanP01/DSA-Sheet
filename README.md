# 60-Day Java DSA Roadmap

## Goal

Learn Data Structures and Algorithms from scratch using Java in **60
days**, progressing from beginner to advanced with a focus on coding
interviews.

------------------------------------------------------------------------

# Phase 1: Java & Arrays (Days 1--10)

## Day 1 -- Java Basics Revision

-   Variables
-   Data types
-   Operators
-   Input/Output
-   Loops
-   Methods
-   Time & Space Complexity (Big O)

**Practice** - Print patterns - Sum of numbers - Prime number -
Factorial - Fibonacci

## Day 2 -- Arrays Basics   

Learn: - Array declaration - Traversal - Insertion - Deletion -
Searching

Problems: - Largest element - Smallest element - Second largest -
Reverse array - Check sorted

## Day 3 -- Array Interview Problems

Problems: - Move zeros - Rotate array - Remove duplicates - Missing
number - Frequency count

## Day 4 -- Prefix Sum

Learn: - Prefix Sum - Running Sum

Problems: - Range sum query - Pivot index - Equilibrium index

## Day 5 -- Sliding Window

Learn: - Fixed window - Variable window

Problems: - Maximum sum subarray - Longest substring - Maximum
consecutive ones

## Day 6 -- Two Pointers

Problems: - Two Sum - Remove duplicates - Container with most water -
Trapping rainwater (intro)

## Day 7 -- Binary Search

Learn: - Binary Search - Lower Bound - Upper Bound

Problems: - Search insert - First occurrence - Last occurrence

## Day 8 -- Binary Search Advanced

Problems: - Search in rotated array - Peak element - Square root - Koko
Eating Bananas

## Day 9 -- Sorting

Learn: - Bubble Sort - Selection Sort - Insertion Sort - Merge Sort -
Quick Sort

Understand: - Stable vs Unstable - In-place vs Out-of-place

## Day 10 -- Revision

Solve 15 mixed array questions.

------------------------------------------------------------------------

# Daily Routine (2--3 Hours)

  Time     Activity
  -------- -----------------------------------------------------
  20 min   Learn theory
  20 min   Watch/read one explanation
  40 min   Implement the data structure/algorithm from scratch
  60 min   Solve 3--5 problems
  20 min   Review mistakes and write notes

------------------------------------------------------------------------

# Learning Rules

1.  Learn the theory first.
2.  Think about the approach before coding.
3.  Write the solution yourself.
4.  Review time and space complexity.
5.  Solve Easy → Medium → Hard problems.
6.  Revise every week.
7.  Focus on understanding patterns, not memorizing solutions.

------------------------------------------------------------------------

# Final Goal

By the end of 60 days, you should be comfortable solving most coding
interview questions involving: - Arrays - Strings - Recursion - Linked
Lists - Stack & Queue - Hashing - Trees - Heap - Graphs - Dynamic
Programming - Advanced Data Structures
